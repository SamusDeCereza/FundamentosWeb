En este repositorio se entuentra el proyecto final correspondiente al 
modulo 2 "Fundamentos del Desarrollo Web", del curso FrontEnd en la 
plataforma Campus propiedad de DevF.

Dicho proyecto esta realizado con CSS, JavaScript y HTML, que en conjunto
crean una pagina dinamica, como parte de mi aprendizaje y reforzamiento en 
conocimientos de desarrollo web. 

--

## Limites

Tomando en cuenta que el proyecto se realiza con las tecnologias base, sin uso de frameworks o herramientas,
es claro que un desarrollo responsive para todos los dispositivos y que se ajuste perfectamente a cada uno
de ellos, es una tarea sumamente tardada, por lo que unicamente se encuentra la vista de escritorio y movil.




##  Proximos pasos

Como proximos pasos, sugiero el uso de grid en maquetar la pagina, y el uso de una libreria para los estilos, tal como Bootstrap o TailWindCSS, que facilite la aplicacion de reglas de estilo y permita un desarrollo mas eficaz.
